# test-bun

> Teil des Monorepos – Pfad: `apps/test-bun`

## Übersicht

Bun-basierte Applikation mit TypeScript.

## Voraussetzungen

- [Bun](https://bun.sh/) ≥ 1.0

## Entwicklung

```bash
# Abhängigkeiten installieren
cd apps/test-bun
bun install

# Starten
bun run src/index.ts

# Tests ausführen
bun test
```
